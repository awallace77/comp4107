# COMP4107: Neural Networks - Assignment 1

## Setup

To setup a virtual python environment to install the pytorch dependency, use the following:

`cd a1`

`python3 -m venv .venv`

`source .venv/bin/activate`

I believe on Windows the command above is:

`source .venv/Scripts/activate`

`pip install -r requirements.txt`

- If using VSCODE, select your interpreter via `ctrl + Shift + p` and select "Python: Select interpreter"
- Select "Enter interpreter path"
- If on Linux: navigate to and select `.venv/bin/python3` in the `a1` directory
- If on Windows: Navigate to and select `.venv/Scripts/python3` in the `a1` directory
